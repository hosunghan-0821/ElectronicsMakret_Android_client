# Used Electronics market (개인 프로젝트)

## 프로젝트 소개 및 특징

- 전자기기를 중고거래 할 수 있는 어플리케이션 입니다.
- 중고거래 어플리케이션(당근마켓, 번개장터, 중고나라)  참고하여 클론코딩 하였습니다.
- 각 플랫폼의 장단점을 가져와 반영 (지역기반 거래 + 택배 안전거래)
- 1대1 영상통화 기능 추가 (중고물품 거래 전 확인가능)

## 📅 프로젝트 기간
'2022-01 - '2022-04 (약 13주~14주)
